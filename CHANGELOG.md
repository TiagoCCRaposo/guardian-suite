## [0.1.0] - 2024-01-XX
### Adicionado
- **Dashboard Interativo:** Interface web desenvolvida com React, Vite e Tailwind CSS.
- **Monitorização de Recursos:** Visualização em tempo real de uso de CPU, Memória RAM e Disco.
- **Scan de Vulnerabilidades:** Módulo base para verificação de falhas de segurança em servidores Linux.
- **Sistema de Relatórios:** Geração de logs detalhados sobre as análises efetuadas.
- **Autenticação:** Estrutura base para controlo de acesso (se aplicável).
- **Backend:** API desenvolvida em Bun para alta performance.
- **Base de Dados:** Integração inicial com PostgreSQL.
